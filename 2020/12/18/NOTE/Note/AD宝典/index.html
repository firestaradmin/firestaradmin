走线：CTRL + W

对齐到栅格：CTRL + SHIFT + D

器件翻转：选中该器件，拖动，鼠标呈十字状，X 键水平左右翻转；Y 键为垂直上下翻转。

批量标注：点击工具 -> 标注 -> 原理图标注

设置Net 作用范围：

Projct（工程） -> Project Option（工程参数） -> Options标签，在Net Identifier Scope(网络标示符范围)中选择。

Net Label 的四种作用范围：

  • “Automatic”是缺省选项，表示系统会检测项目图纸内容，从而自动调整网络标识的范围。检测及自动调整的过程如下：
    • 如果原理图里有Sheet Entry标识，则网络标识的范围调整为Hierarchical。如果原理图里没有Sheet Entry标识。但是有Port标识，则网络标识的范围调整为Flat。如果原理图里既没有Sheet Entry标识，又没有Port标识，则Net Label的范围调整为Global。
  • “Flat”代表扁平式图纸结构，这种情况下，Net Label的作用范围仍是单张图纸以内。而Port的作用范围扩大到所有图纸，各图纸只要有相同的Port名，就可以发生信号传递。
  • “Hierarchical”代表层次式结构，这种情况下，Net Label，Port的作用范围是单张图纸以内。当然，Port可以与上层的Sheet Entry连接，以纵向方式在图纸之间传递信号。
  • “Global”是最开放的连接方式，这种情况下，Net Label、Port的作用范围都扩大到所有图纸。各图纸只要有相同的Port或相同的Net Label，就可以发生信号传递。

PCB

切换单位：Q

层 视图选项：L

网络高亮：CTRL + 鼠标单击需要高亮的网络

层显示模式 [单层、多层、其它层灰色模式]：SHIFT+ S

更改走线模式：SHIFT+ 空格

翻转选中的元件：M + I

翻转电路板：CTRL + F

切换布线层：NUM* | L

布线时切换层并放置过孔：CTRL + SHIFT+ 滚轮

布线过程中放置过孔：NUM* | 2(大键盘上的数字2)

哪里要测点哪里：CTRL + M

测量边距：R + P

选中整条线：CTRL + H

切换线宽：SHIFT+ w | 3

切换栅格大小：G

忽略栅格捕捉：按住CTRL

快速布局选中的元器件：选中元件后 T + O + L 左键拖拽

固定间距的等间距走线：T + T + M

可更改间距的等间距走线：P + M

走线时显示走线长度：SHIFT+ G

显示或关闭坐标显示信息：SHIFT+ H

显示或关闭放大镜：SHIFT+ M

局部自动走线：SHIFT+ A

复位错误标志：T + M

PCB 设计时的技巧：

  • PCB设计时，交叉选择是个好东西
  • PCB设计时，可为网络建立Net Class 分组，来设计规则，和显示隐藏飞线
  • PCB设计时，分组的网络显示和隐藏，在右下角panels 里的PCB框设置
  • PCB设计时，设置好规则 记得复位错误标志 快捷键 T + M
